## Project Status

### Token Status:
- The **AXNV** token (Axionova) has been successfully launched, with a total supply of **750 million tokens**. The token contract address is [0xaFdEC2E212E771B758C75bE31f14F008F158DED7](https://bscscan.com/token/0xaFdEC2E212E771B758C75bE31f14F008F158DED7).

- The smart contract for **AXNV** has been developed and deployed. It has passed an audit using **Slither** for vulnerability detection, with the audit report available in the project's **[GitHub repository](https://github.com/Axionova-coin/Axionova/blob/main/audits/slither-report.md)**.
